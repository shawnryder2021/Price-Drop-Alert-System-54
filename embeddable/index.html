Price Drop Alert Widget

A lightweight, customizable widget for your dealership website

How It Works

This demo page shows the Price Drop Alert Widget in action. You can see the floating button in the bottom-left corner of this page.

Try it out: Click the "Price Drop Alerts" button to see the widget in action!

Easy Installation

Add this single line of code to your website to install the widget:

<script src="widget.js" async></script>

Customization

Open the widget.js file and modify these settings at the top:

const config = { dealershipName: "Premium Auto Gallery", // Your dealership name webhookUrl: "https://your-webhook-url.com", // Your webhook URL accentColor: "#3b82f6", // Primary color position: "bottom-left", // Position on screen buttonText: "Price Drop Alerts", // Button text logoUrl: "" // Optional logo URL };

Works Anywhere

The widget works on any website platform:

  • WordPress
  • Shopify
  • Wix
  • Squarespace
  • Custom HTML websites
  • Dealer-specific platforms
